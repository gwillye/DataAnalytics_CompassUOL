# AWS Partner: Data Analytics on AWS - Business

## Módulo 1 - Introdução ao curso e oportunidade de mercado

Antes de tudo, vale fixar o conceito de Big Data: são dados em grande volume, com criação e crescimento rápidos e uma variedade de informações verossímeis, de forma que armazenar e gerenciar esses dados acaba virando um desafio.

### Dados como um ativo estratégico e diferencial

Quando falamos de dados, estamos falando de informações registradas em formato digital, que podem ser transmitidas, processadas, analisadas e armazenadas. Esses dados se transformam em insights, geram valor econômico e permitem oferecer serviços altamente personalizados. Não à toa, "Data is the new oil" é uma frase popular que define os dados como o recurso mais valioso do mundo.

Os dados são ativos estratégicos valiosos, e as empresas costumam usá-los seguindo um processo bem simples:

Coleta dos dados -> Transformação dos dados em insights -> Disponibilização dos dados para usuários e clientes relevantes -> Criação de novos produtos e serviços -> Investimento em tecnologias de processamento de dados.

### Oportunidades de mercado para dados modernos e análise

Mesmo com todas as vantagens, e mesmo sendo um ativo muito valioso, os dados trazem alguns desafios na hora de fazer uma boa análise. Os principais são:

* Conhecimento
* Habilidade
* Segurança
* Privacidade

Sabendo desses desafios, surge uma oportunidade. Ao ajudar os clientes a inovar e a obter valor, você pode auxiliá-los a extrair valor de dados que crescem exponencialmente, vêm de novas fontes, são cada vez mais diversos, usados por muitas pessoas e analisados por muitas aplicações.

Os clientes costumam ter dificuldades com os data warehouses tradicionais, por causa da falta de comunicação entre silos, da dificuldade de analisar e acessar os dados, do alto custo de manutenção e da dificuldade de escalar (além de serem caros). Além dos insights, eles querem analisar e extrair informações em tempo real, e é aí que entra a utilidade de criar e usar data lakes.

Data lakes são repositórios centrais que permitem armazenar dados estruturados e não estruturados em qualquer escala. Seus elementos essenciais são:

* Movimentação de dados
* Armazenamento e catalogação de dados com segurança
* Análise
* Machine Learning

O Data Lake agrega valor para o cliente de várias formas: melhor tomada de decisão, melhores interações com os clientes, melhores escolhas de inovação em P&D e mais eficiência operacional.

### Pipeline de dados

Os componentes do pipeline de dados são:

* Coletar
* Armazenar
* Processar
* Analisar
* Extrair insights acionáveis

Os clientes enfrentam alguns desafios em relação ao pipeline de dados. Eles precisam coletar uma variedade de tipos de dados acumulados em velocidades variáveis, coletar dados de várias fontes que chegam em ritmos diferentes, armazenar grandes quantidades de dados sem esgotar o espaço e ainda limpar e aumentar a qualidade dos dados que serão analisados.
